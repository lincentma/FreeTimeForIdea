- [ ] 如何快速对比数据库表结构的差异?
- 背景：会存在测试环境数据库字段类型、字段默认值、字段描述与正式环境数据库不一致的情况，在上线前如果可以快速Diff并及时修改保持一致，可以避免很多问题。
- 输入：测试环境数据库MySQL链接地址 + 正式环境数据库MySQL链接地址
- 输出：Diff情况以及对应SQL语句
- 产品形态：API接口形式？Diff情况接口返回？
